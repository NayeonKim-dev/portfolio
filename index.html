About

저는 SEO와 웹 접근성을 고려하여 시멘틱 태그를 활용한 마크업으로 사용자 경험을 최우선으로 삼는 웹 퍼블리셔입니다. 시각적인 즐거움을 제공하는 동시에, 모든 사용자들이 콘텐츠에 원활하게 접근하고 이용할 수 있도록 노력합니다. 동료와 원활한 커뮤니케이션을 바탕으로 팀 프로젝트를 효율적으로 이끌어내며, 꾸준한 학습과 개발 업데이트를 통해 더 나은 웹 경험을 제공하기 위해 노력합니다.

skill

html

시멘틱 태그를 활용하여 웹 표준과 웹 접근성을 준수해 코드를 작성하며 크로스 브라우징으로 호환성을 고려합니다. 또한 구조에 적합한 클래스명으로 웹 문서를 작성할 수 있습니다.

css

미디어 쿼리를 이용해 반응형 웹사이트를 구현할 수 있으며 그리드와 플렉스 박스 속성을 사용해 유연하고 정확한 페이지 레이아웃을 만들 수 있습니다. 또한 웹 폰트와 이미지 스프라이트를 사용해 용량을 최적화 해 사용자 경험을 개선할 수 있습니다.

jquery

jQuery를 활용하여 동적 웹 페이지를 구축할 수 있으며 json을 통한 비동기 데이터 로딩을 구현할 수 있습니다. 또한 추상화를 통해 보다 효율적으로 코드를 작성할 수 있습니다.

gsap

GSAP을 활용하여 웹사이트에 동적 애니메이션 효과를 구현할 수 있습니다. 타임라인, 스크롤 애니메이션, 콜백 함수 등을 적용하여 사용자에게 시각적으로 풍부하고 흥미로운 경험을 제공할 수 있습니다.

swiper

Swiper 라이브러리의 다양한 옵션을 활용하여 웹 페이지의 시각적인 효과를 극대화하고 인터랙티브한 슬라이드를 구현할 수 있습니다.
